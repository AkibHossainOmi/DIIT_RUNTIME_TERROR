### [Run Backend](./backend/README.md)
Go to the project directory `DIIT_RUNTIME_TERROR/` and open terminal.
<br>
Enter the following command,
```
cd backend
npm i
node server.js
```

### [Run Frontend](./frontend/README.md)
Go to the project directory `DIIT_RUNTIME_TERROR/` and open terminal.
<br>
Enter the following command,
```
cd frontend
npm i
npm start
```
